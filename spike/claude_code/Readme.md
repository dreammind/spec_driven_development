## 自分用のメモ

vscode + claude code 拡張を使って、下記の流れで仕様駆動型開発+テスト駆動型開発のお試しを実施した

1. [事前の要件定義書](doc/PreRequirement.md)を作成した
1. `claude code`に`事前の要件定義書`をもとに[要件定義書](doc/Requirement.md)を作成してと依頼
1. [要件定義書](doc/Requirement.md)に検索機能を追加してもらった
1. [要件定義書](doc/Requirement.md)に次の目的として、[設計書](doc/Design.md)を作成してと追加
1. 設計書の削除機能が物理削除だったので、論理削除に変更した
1. 設計書を元に[実装計画](doc/ImplementationPlan.md)を作成してもらった
1. また、実装計画を元に、一般的なユーザ向けの[利用説明書](doc/Usage.md) を作成してもらった
1. 実装計画に基づいてテストと実装を進めてと依頼
1. 実装が完了した後、[セッション履歴](doc/SessionHistory.md)を作成してもらった
