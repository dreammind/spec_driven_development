# 要件定義書 — CLIタスク管理（TODO）アプリ

## 次の目的

本要件定義書をもとに、設計書（[doc/Design.md](doc/Design.md)）を作成してください。

## 1. 概要

本ドキュメントは、コマンドラインインターフェース（CLI）で動作するタスク管理アプリケーションの要件を定義します。

---

## 2. 目的

ユーザーが日常のタスクをコマンドラインから効率的に管理できるツールを提供します。タスクには優先度・期限・カテゴリを設定でき、柔軟な分類・絞り込みが可能です。

---

## 3. ユーザー

- CLI操作に慣れた個人ユーザー
- シングルユーザー環境を前提とする

---

## 4. 機能要件

### 4.1 タスク操作

| 機能 | 説明 |
|---|---|
| タスク追加 | タイトル、優先度、期限、カテゴリを指定してタスクを作成できる |
| タスク完了 | 指定したタスクを完了状態にマークできる |
| タスク削除 | 指定したタスクを削除できる |
| タスク一覧表示 | 登録済みのタスクを一覧で表示できる |
| タスク詳細表示 | 指定したタスクの詳細情報を表示できる |
| タスク編集 | 登録済みタスクのタイトル・優先度・期限・カテゴリを変更できる |

### 4.2 優先度

- 優先度は「高・中・低」の3段階で設定できる
- 優先度未指定の場合はデフォルト値（中）が適用される

### 4.3 期限

- タスクに期限（日付）を設定できる
- 期限が設定されているタスクは、期限切れかどうかを判別して表示できる
- 期限は任意項目とする

### 4.4 カテゴリ

- タスクに任意のカテゴリ名を設定できる（例：仕事、プライベート、買い物）
- カテゴリは複数作成可能
- カテゴリは任意項目とする

### 4.5 検索

- タイトルに含まれるキーワードでタスクを検索できる
- 検索はタイトルの部分一致で行う
- 検索結果は一覧形式で表示される
- 検索キーワードに一致するタスクが存在しない場合はその旨を表示する

### 4.6 フィルタリング・並べ替え

- 以下の条件でタスクを絞り込み表示できる
  - 完了 / 未完了
  - 優先度
  - カテゴリ
  - 期限切れ
- 以下の条件で並べ替えができる
  - 優先度順
  - 期限順
  - 作成日順

---

## 5. 非機能要件

### 5.1 データ保存

- データはローカルファイル（`tasks.json` または `tasks.yaml`）に保存する
- 外部データベースは使用しない
- アプリ終了後もデータが永続化される

### 5.2 品質

- 型チェックによる安全性を確保する
- 自動テストを導入する
- テストカバレッジ 90% 以上を目標とする

### 5.3 ユーザビリティ

- コマンドのヘルプメッセージを提供し、使い方をユーザーが確認できる
- 不正な入力に対して、わかりやすいエラーメッセージを表示する

---

## 6. 制約事項

- 外部データベース（RDB、クラウドDBなど）は使用しない
- ローカルファイルストレージのみ使用可能
- シングルユーザー利用を前提とし、同時アクセス制御は不要

---

## 7. データ項目

タスクは以下の情報を持つ：

| 項目 | 必須 | 説明 |
|---|---|---|
| ID | 必須 | タスクを一意に識別する識別子 |
| タイトル | 必須 | タスクの内容を表す文字列 |
| 完了フラグ | 必須 | 完了・未完了の状態 |
| 優先度 | 必須 | 高・中・低の3段階（デフォルト：中） |
| 期限 | 任意 | タスクの締め切り日 |
| カテゴリ | 任意 | タスクを分類するラベル |
| 作成日時 | 必須 | タスクが作成された日時 |
| 更新日時 | 必須 | タスクが最後に更新された日時 |

---

## 8. ユースケース

### UC-01: タスクを追加する
- ユーザーはタイトルを必須入力し、任意で優先度・期限・カテゴリを指定してタスクを追加できる

### UC-02: タスク一覧を確認する
- ユーザーはすべてのタスクを一覧で確認できる
- 完了済み・未完了でフィルタリングできる

### UC-03: タスクを完了にする
- ユーザーはタスクIDを指定して、そのタスクを完了状態にできる

### UC-04: タスクを削除する
- ユーザーはタスクIDを指定して、そのタスクを削除できる

### UC-05: タスクを編集する
- ユーザーはタスクIDを指定して、タイトル・優先度・期限・カテゴリを変更できる

### UC-06: タスクを絞り込む
- ユーザーはカテゴリ・優先度・完了状態・期限切れでタスクを絞り込んで表示できる

### UC-07: タスクを検索する
- ユーザーはキーワードを入力し、タイトルに部分一致するタスクを一覧で確認できる
- 一致するタスクがない場合はその旨がメッセージで表示される
