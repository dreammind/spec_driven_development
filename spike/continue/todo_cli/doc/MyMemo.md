 # 自分用のメモ

 * Continue をどのようなモードで使うのか
	•	Chat：ツールなし（純会話） ￼
	•	Plan：読み取り系ツールのみ（安全に調査・計画） ￼
	•	Agent：読み取り＋編集/作成など“変更系ツール”も使って実装まで進める ￼

  -  Chat の使いどころ（最安・最速・安全）
      目的：相談・方針・概念整理（コードを触らせない）
  - Plan の使いどころ（安全な“調査と設計”）
      目的：既存コードを調べさせて、実装手順を固める（まだ書かせない）
  - Agent の使いどころ（実装を“やらせる”）
      目的：複数ファイル編集・追加・修正・コマンド実行など、作業を進めさせる
