# ユーザーストーリー

- CLIユーザーとして、タスクを追加したい。そうすることで、やるべきことを記録できる。
- CLIユーザーとして、タスクに優先度を付けたい。そうすることで、重要度に応じて着手できる。
- CLIユーザーとして、タスクに期限を付けたい。そうすることで、締切に間に合うように管理できる。
- CLIユーザーとして、タスクにカテゴリを付けたい。そうすることで、用途ごとに整理できる。
- CLIユーザーとして、タスク一覧を見たい。そうすることで、全体の進捗を把握できる。
- CLIユーザーとして、完了/未完了でタスクを絞り込みたい。そうすることで、今やるべきことに集中できる。
- CLIユーザーとして、優先度・期限・カテゴリでタスクを絞り込みたい。そうすることで、条件に合うタスクを素早く見つけられる。
- CLIユーザーとして、期限順や優先度順に並べ替えたい。そうすることで、優先度に沿った作業計画を立てられる。
- CLIユーザーとして、タスクを完了にしたい。そうすることで、達成状況を明確にできる。
- CLIユーザーとして、完了済みタスクを未完了に戻したい。そうすることで、誤操作を修正できる。
- CLIユーザーとして、タスク内容を編集したい。そうすることで、状況変化に対応できる。
- CLIユーザーとして、タスクを削除したい。そうすることで、不要な情報を整理できる。
- CLIユーザーとして、キーワードでタスクを検索したい。そうすることで、目的のタスクをすぐ見つけられる。
